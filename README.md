# Liga MX Goal Scoring Table scraper

1. Create virtual environment.
    ```cmd
    py -m venv venv
    ```
1. Open virtual environment.
    ```cmd
    .\venv\Scripts\activate
    ```
1. Install all dependencies.
    ```cmd
    pip install -r requirements.txt
    ```
1. Run main.py file
    ```cmd
    python main.py
    ```
